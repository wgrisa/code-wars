# count-by-x
// https://www.codewars.com/kata/count-by-x/solutions/javascript/me/best_practice


```
function countBy(x, n) {
  var z = [];
  for (var i = 1; i <= n; i++) {
    z[i-1] = x*i;
  }
  return z;
}
```
